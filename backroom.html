GIT COMMAND REFERENCE

                    
                    

                        
SETUP & HELP

                        

                            git config --global user.name "Name"
                            
Set your global Git author name

                        

                        

                            git config --global user.email "you@email.com"
                            
Set your global Git email

                        

                        

                            git help <command>
                            
Open help for a specific command

                        

                    


                    

                        
REPO BASICS

                        

                            git init
                            
Initialize a new repository

                        

                        

                            git clone <url>
                            
Clone a remote repository

                        

                        

                            git status
                            
Show working tree status

                        

                        

                            git add <file>
                            
Stage changes for commit

                        

                        

                            git commit -m "message"
                            
Commit staged changes with a message

                        

                    


                    

                        
BRANCHING

                        

                            git branch
                            
List local branches

                        

                        

                            git branch <name>
                            
Create a new branch

                        

                        

                            git switch <name>
                            
Switch to another branch

                        

                        

                            git switch -c <name>
                            
Create and switch to a new branch

                        

                        

                            git checkout <name>
                            
Legacy branch switch command

                        

                    


                    

                        
SYNC WITH REMOTE

                        

                            git remote -v
                            
List remotes and their URLs

                        

                        

                            git fetch
                            
Download remote updates without merging

                        

                        

                            git pull
                            
Fetch and merge remote changes

                        

                        

                            git push
                            
Push local commits to remote

                        

                    


                    

                        
HISTORY & INSPECTION

                        

                            git log
                            
Show commit history

                        

                        

                            git log --oneline --graph --decorate
                            
Compact history with branch graph

                        

                        

                            git diff
                            
Show changes not yet staged

                        

                        

                            git diff --staged
                            
Show staged changes

                        

                        

                            git show <commit>
                            
Show details for a specific commit

                        

                    


                    

                        
UNDO & FIX

                        

                            git restore <file>
                            
Discard changes in working tree

                        

                        

                            git restore --staged <file>
                            
Unstage a file

                        

                        

                            git reset --soft HEAD~1
                            
Undo last commit, keep changes staged

                        

                        

                            git reset --hard HEAD~1
                            
Undo last commit and discard changes

                        

                        

                            git revert <commit>
                            
Create a new commit that reverses changes

                        

                    


                    

                        
MERGE & CONFLICT RESOLUTION

                        

                            git merge <branch>
                            
Merge specified branch into current branch

                        

                        

                            git merge --abort
                            
Abort the merge and return to pre-merge state

                        

                        

                            git merge --continue
                            
Continue merge after resolving conflicts

                        

                        

                            git mergetool
                            
Launch visual merge tool

                        

                        

                            git checkout --ours <file>
                            
Accept current branch version

                        

                        

                            git checkout --theirs <file>
                            
Accept incoming branch version

                        

                        

                            git add <file>
                            
Mark conflict as resolved

                        

                        

                            git log --merge
                            
Show commits that caused the conflict

                        

                    


                    

                        
STASH & CLEANUP

                        

                            git stash
                            
Temporarily save uncommitted changes

                        

                        

                            git stash list
                            
List saved stashes

                        

                        

                            git stash pop
                            
Apply and remove latest stash

                        

                        

                            git clean -fd
                            
Remove untracked files and folders

                        

                    


                    

                        
TAGS & RELEASES

                        

                            git tag
                            
List tags

                        

                        

                            git tag -a v1.0.0 -m "release"
                            
Create an annotated tag

                        

                        

                            git push --tags
                            
Push tags to remote
